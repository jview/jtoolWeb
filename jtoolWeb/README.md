将jtools java常用工具集发布成web以便于进行交互式使用
===================================

设计目标
-----------------------------------
设计目标是成为一个可扩展的不依赖现有系统，能直接使用的第三方工具集，并可配合系统命令，如dos/linux命令，执行相关命令或任务，如数据库查询，或数据库数据导出。
